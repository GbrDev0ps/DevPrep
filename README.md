## 🚀 Sobre

O **DevPrep** é um aplicativo desenvolvido para auxiliar desenvolvedores iOS na preparação para entrevistas técnicas.

O aplicativo reúne perguntas frequentes utilizadas em processos seletivos, organizadas por nível de experiência, além de questões comportamentais e simulados completos para praticar em um ambiente semelhante ao de uma entrevista real.

O objetivo é oferecer uma ferramenta simples, rápida e intuitiva para estudar em qualquer lugar.

---

## ✨ Funcionalidades

- 📚 Perguntas técnicas para iOS
- 🎯 Simulados completos
- 👨‍💻 Perguntas divididas por senioridade
  - Junior
  - Pleno
  - Sênior
- 💬 Perguntas comportamentais
- 🔍 Busca de perguntas
- 📂 Categorias
- ⚡ Interface rápida e intuitiva
- 🌙 Design moderno utilizando SwiftUI

---

## 🏗️ Arquitetura

O projeto foi desenvolvido seguindo a arquitetura **MVVM (Model-View-ViewModel)**, buscando uma separação clara de responsabilidades, facilidade de manutenção e escalabilidade.

```
Views
   │
ViewModels
   │
Models
```
